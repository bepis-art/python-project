# Проект "Трекер привычек"
Телеграм-бот для формирования и поддержания полезных привычек с автоматическими напоминаниями, подтверждением выполнения и статистикой


## Бот помогает пользователям выработать устойчивые привычки через регулярную практику:
- Добавление привычек с гибким интервалом (от 1 минуты до 24 часов)
- Автоматические напоминания через заданный интервал
- Подтверждение выполнения командой /done (в течение 1 минуты после напоминания)
- Статистика: прогресс по каждой привычке и общая за неделю
- Управление: включение/выключение, удаление привычек, сброс статистики
- Изоляция данных: каждый пользователь видит только свои привычки


## Стек:
- Python 3.11
- FastAPI (архитектурная основа)
- python-telegram-bot 
- SQLAlchemy
- PostgreSQL (open-source)
- Celery
- Redis
- Docker + Docker Compose
- pytest


## Команды: 
- /start - Регистрация
- /add_habit <описание> <интервал> - Добавить привычку (интервал в минутах)
- /habits - Список привычек со статистикой
- /done - Подтвердить выполнение последнего напоминания
- /stats - Статистика за последнюю неделю
- /pause_habit <номер> - Приостановить привычку
- /resume_habit <номер> - Возобновить привычку
- /delete_habit <номер> - Удалить привычку и её статистику
- /reset_stats - Обнулить всю статистику


## Как запустить? 
- git clone -b final https://github.com/bepis-art/python-project.git
- cd habit-tracker-bot
- Создать .env:
-- TELEGRAM_BOT_TOKEN=token
-- DATABASE_URL=postgresql://habituser:habitspass@db:5432/habitdb
-- CELERY_BROKER_URL=redis://redis:6379/0
-- CELERY_RESULT_BACKEND=redis://redis:6379/0
- docker-compose up --build -d

## Запуск тестов:
- Запустить бота
- docker-compose run --rm bot python -m pytest tests/ -v -s
